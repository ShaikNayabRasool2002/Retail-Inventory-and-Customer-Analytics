
# 📊 Retail, Inventory & Customer Analytics Dashboard

## 📌 Overview

This project is an interactive **Power BI dashboard** designed to analyze **retail performance, inventory management, and customer behavior**. It helps businesses make data-driven decisions by providing insights into sales trends, stock levels, and customer segmentation.

---

## 🎯 Objectives

* Monitor overall **sales performance**
* Track **inventory levels** and stock movement
* Analyze **customer purchasing behavior**
* Identify **top-performing products and categories**
* Improve decision-making with visual insights

---

## 📂 Dataset

The dataset includes:

* 🛒 Sales transactions
* 📦 Inventory details
* 👥 Customer information
* 📅 Time-based data (daily/monthly/yearly trends)

---

## 📊 Key Features

### 🔹 Sales Analysis

* Total Revenue & Profit
* Sales trends over time
* Region-wise and category-wise performance

### 🔹 Inventory Analysis

* Stock availability
* Fast-moving vs slow-moving items
* Inventory turnover insights

### 🔹 Customer Analytics

* Customer segmentation
* Purchase patterns
* Top customers by revenue

---

## 📈 Dashboard Highlights

* Interactive filters and slicers
* Dynamic visuals (charts, KPIs, maps)
* Drill-down capabilities for deeper insights

---

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* Data modeling using relationships
* DAX (Data Analysis Expressions)
* Data cleaning & transformation (Power Query)

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it in **Power BI Desktop**
3. Explore dashboards using filters and slicers
4. Interact with visuals for detailed insights
